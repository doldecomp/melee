## Building

### Requirements

- **[DevKitPro]**

- **[Python3]**

    ```sh
    pacman -S msys/python3
    ```

- **[GCC]**

    ```sh
    pacman -S gcc
    ```

### Instructions

1. Download <kbd>**[GC_WII_COMPILERS.zip][GC]**</kbd>

2. Extract it's content to `/tools/mwcc_compiler/`.

3. Run the **Make** command :

    ```sh
    make
    ```
